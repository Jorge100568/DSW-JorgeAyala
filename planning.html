CONTENIDO DE HTML5


OBJETIVO GENERAL

Se desarrolla el OVI del tema contenido HTML5 (SECTION, ASIDE, FOOTER, ARTICLE) para el curso de diseños web buscado que el estudiante mediante la estrategia de aprendizaje de recursos virtuales de la UNAD tenga claro los elementos de diseño Etiquetas Semánticas para la elaboración de una pagina web y un MOCKUP


OBJETIVOS ESPECIFICOS

  • Dar a conocer la importancia de los elementos estructurales de HTML5 al estudiante
  • Presentar definicion e implementacion de las etiquetas semanticas de HTML5 (SECTION, ASIDE, FOOTER, ARTICLE)
  • Presentar el diseño del MOCKUP de una página principal

CONTENIDO

Bienvenidos al curso virtual de diseños Web, veremos la temática de los contenidos de HTML5 donde se les explicara las etiquetas semánticas como (SECTION, ASIDE, FOOTER, ARTICLE) estructura principal para la realización de una página web y una maquetación MOCUKP del proyecto a entregar, obteniendo dominio básico en el lenguaje el curso busca que el estudiante fortalezca y se apropie de conceptos aplicándolos para situaciones reales del mundo laboral.
